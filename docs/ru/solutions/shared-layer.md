# Слой shared

## Семантика
Набор модулей общего или инфроструктурного назначения. Cамописные библиотеки, которые не целесообразно выносить в npm пакеты.

#### Требования
- не содержит модулей связанных с бизнес-логикой

#### Рекомендации
- минимизировать взаимодействие модулей друг с другом внутри shared

#### Свойства
- могут быть использованы в любом месте проекта

Модули могут быть как stateless, так и stateful.

## Синтаксис

Структура папок:
```
src/
└── shared/
    ├── module-1/
    │   ├── ...
    │   └── index.ts
    ├── module-2/
    │   ├── ...
    │   └── index.ts
    └─...
```

## Автоматизация
запретить использование файлов в папке shared
